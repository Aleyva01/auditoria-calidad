⚡ Paso 1 — Conectar Google Apps Script (obligatorio para guardar)

Esto permite que los registros se guarden en Google Sheets desde cualquier dispositivo sin restricciones.

1

Abre tu Google Sheet de respuestas → menú Extensiones → Apps Script.

2

Borra todo el código que aparece y pega exactamente esto:

const SHEET_ID = '1DTV6rFCwIq1NzAU7pjMfuvgnt_ep-4veuDU0Vi2Fys4'; const SHEET_NAME = 'Respuestas de formulario 1'; function doPost(e) { try { const data = JSON.parse(e.postData.contents); const sheet = SpreadsheetApp.openById(SHEET_ID).getSheetByName(SHEET_NAME); const now = new Date(); const fecha = Utilities.formatDate(now, 'America/Mexico_City', 'dd/MM/yyyy HH:mm:ss'); sheet.appendRow([ fecha, data.corte || '', data.programa || '', data.modelo || '', data.prec || '', data.prev || '', data.auditor || '', data.maquilero|| '', data.tipo || '', '', data.puntos || '', '', '', '', '', '', '', data.cal || '', data.obs || '', ]); return ContentService.createTextOutput(JSON.stringify({ok:true})) .setMimeType(ContentService.MimeType.JSON); } catch(err) { return ContentService.createTextOutput(JSON.stringify({ok:false,error:err.message})) .setMimeType(ContentService.MimeType.JSON); } } function doGet(e) { return ContentService.createTextOutput('OK'); }
3

Clic en Guardar (ícono de disco 💾) → ponle el nombre AuditoriaScript.

4

Clic en Implementar → Nueva implementación → tipo: Aplicación web → "Ejecutar como": Yo → "Quién tiene acceso": Cualquier persona → clic en Implementar.

5

Copia la URL de implementación que aparece (empieza con https://script.google.com/macros/s/...) y pégala aquí:

Paso 2 — Publicar en GitHub Pages

URL pública permanente accesible desde cualquier dispositivo.

1

Crea cuenta en GitHub — github.com

2

Nuevo repositorio: + → New repository → nombre: auditoria-calidad → Public → Crear.

3

Sube el archivo descargado (después de guardar la URL del Apps Script): Add file → Upload files → renómbralo a index.html → Commit changes.

4

Activa GitHub Pages: Settings → Pages → Source: main / root → Save.

5

Tu URL: https://TU-USUARIO.github.io/auditoria-calidad/
