Publica tu app gratis en GitHub Pages

GitHub Pages te da una URL pública permanente. Cualquier persona con la URL y la contraseña puede acceder desde su celular o computadora.

1

Crea una cuenta gratuita en GitHub
Ve a github.com y regístrate si aún no tienes cuenta.

2

Crea un nuevo repositorio
Clic en el botón + → New repository. Ponle un nombre como auditoria-calidad. Márcalo como Public y da clic en Create repository.

3

Sube el archivo HTML
En tu repositorio, clic en Add file → Upload files. Arrastra el archivo auditoria_producto_terminado.html y clic en Commit changes.
Luego renómbralo: clic en el archivo → ícono de lápiz → cambia el nombre a index.html → Commit changes.

4

Activa GitHub Pages
Ve a Settings (engrane) → sección Pages en el menú izquierdo → en Source selecciona Deploy from a branch → elige main y / (root) → clic en Save.

5

¡Listo! Tu URL será:
https://TU-USUARIO.github.io/auditoria-calidad/
Comparte esa URL con tu equipo. Cada vez que actualices el archivo en GitHub, la app se actualiza automáticamente.

Tip: Cuando actualices la contraseña o la lista de maquileros en Configuración, descarga el archivo actualizado y vuelve a subirlo a GitHub (paso 3).
